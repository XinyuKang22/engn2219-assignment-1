I-Mode Memory Instructions


Stack:
0110 <imm8:start address> <imm8:end address>

pseudo: call lable
~= jp lable ; but using stack register

save stack: label_stack
0110 0000 <imm8:end address>	
	jump to <imm8:end address>

call label_stack:
0110 z001 <imm8:label_stack>
	load the stack from mem
	save <pc+1> <imm8:end address> into stack register
	overwrite pc to label_stack+1
	turn on running mode

stack running mode:
	when pc == <imm8:end address> | stack register
	overwrite pc to <pc+1> | stack register
	turn off running mode
